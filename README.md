# X-FNC

Thank you for your interest in our work! </br>

This is the code for our WSDM 2023 paper "Few-shot Node Classification with Extremely Weak Supervision".

### Requirement:'
  Python == 3.7.10
  torch == 1.8.1
  torch-cluster == 1.5.9
  torch-scatter == 2.0.6
  torch-sparse == 0.6.9
  torch-geometric == 1.4.1
  torch-spline-conv==1.2.1
  numpy == 1.18.5
  scipy == 1.5.3
  cuda == 11.0
  tensorboard == 2.2.2
  networkx == 2.5.1
  scikit-learn == 0.24.1
  pandas==1.2.3'


### Code Running:
Run the command: 
`unzip few_shot_data.zip`

Then run the command:
`python train.py`
